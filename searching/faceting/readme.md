# Faceting

如 [搜索概览](../overview.md) 所述，faceting 是基于被索引词项将搜索结果分类的安排。
搜索器以被索引词项及每个词项所找到的匹配的文档数量来呈现。
Faceting 让用户能够更容易地浏览搜索结果，收缩到他们在所搜索的更准确的结果上。
